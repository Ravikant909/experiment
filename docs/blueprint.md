# **App Name**: SplitzyTip

## Core Features:

- Bill Input: Allow users to input the total bill amount.
- Tip Percentage Selection: Enable users to select a tip percentage (e.g., 15%, 20%, custom).
- Split Calculation: Calculate the tip amount and the total bill amount per person, given the number of people.
- Customizable Splitting: Let the user decide how the bill is split: evenly, or custom amounts.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C7) to evoke a friendly and inviting feel.
- Background color: Light desaturated lavender (#F2EBF9) to provide a calm and subtle backdrop.
- Accent color: Vibrant violet (#9D7CBF) for interactive elements and call-to-action buttons.
- Body and headline font: 'Poppins' sans-serif for a clean, modern, readable appearance.
- Use minimalist icons for different functionalities such as bill input, percentage selection, and split calculation.
- Keep layout simple and intuitive, prioritize a single column design and large tap targets.
- Add subtle animations on user interactions, such as tapping a button or entering values, to provide visual feedback.
